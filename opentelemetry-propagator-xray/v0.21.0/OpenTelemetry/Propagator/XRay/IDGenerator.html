#generate_trace_id ⇒ bytes

Generates a valid trace identifier, a 16-byte string with at least one non-zero byte. AWS Docs: docs.aws.amazon.com/xray/latest/api/API_PutTraceSegments.html hi - 4 bytes timestamp, 4 bytes random (Mid) low - 8 bytes random. Since we include timestamp, impossible to be invalid.
